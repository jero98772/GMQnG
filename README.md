### what do you need
### hardware
* esp8266
* MQ 4
* breadbord
* some wires
### software 
* arduino ide 
* esp8266 board config [how configure ep8266 in arduino ide](https://youtu.be/2DL8FlrBTDs?t=77)
some libraries like 
* ESP8266WiFi
* WiFiClient
* ESP8266WebServer
